## Correr en local

Clonar el proyecto

```bash
  git clone https://github.com/jotredev/adminmkt-free.git
```

Ir al directorio del proyecto

```bash
  cd my-project
```

Instalar dependencias

```bash
  npm install
  # o
  pnpm install
```

Iniciar el servidor en local

```bash
  npm run dev
  # o
  pnpm dev
```
